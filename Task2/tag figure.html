Элементы <figure> и <figcaption>

Элемент <figure> используется в сочетании с <figcaption> и предназначен для разметки
диаграмм, иллюстраций, фотографий, примеров кода и других вещей. Этот элемент исполь-
зуется для группирования любых элементов, например, изображений и подписей к ним.
Элемент <figcaption> представляет собой заголовок или описание для <figure>. Данный
элемент не обязателен и добавляется до или после содержимого внутри <figure>. Только
один <figcaption> может находиться внутри <figure>, при этом сам элемент <figure>
может содержать несколько дочерних элементов (например, <img> или <code>).

Использование <figure> и <figcaption>

  1. <figure> для изображения

    2. Изображение в элементе <figure> без подписи:

    Код для этой картинки такой:

    <figure>
    <img src="cute-cat.jpg" width="215" height="200"
    title="Cute Cat" alt="Cute Cat"/>
    </figure>

  2. <figure> с изображением и подписью

    3. Изображение внутри элемента <figure> с поясняющей подписью:

    Код для этой картинки такой:

    <figure>
    <img src="cats.jpg" width="215" height="200"
    title="Cats" alt="Cats"/>
    <figcaption>Котики
    </figcaption>
    </figure>

  3. <figure> с несколькими изображениями

    Размещение нескольких изображений внутри одного элемента <figure>
    с общей подписью:

    Код для этой картинки такой:

    <figure>
    <img src="cats2.jpg" width="70" height="65"
    title="yellow Cat" alt="yellow Cat"/>
    <img src="cats3.jpg" width="90" height="50"
    title="blue Cat" alt="blue Cat"/>
    <img src="cats4.jpg" width="70" height="65"
    title="orange Cat" alt="orange Cat"/>
    <figcaption>Три Котика
    </figcaption>
    </figure>

Плюсы использования

Можно использовать для блоков кода, аудио, видео, графики (возможно, с использованием
<canvas> <svg>), стихов или таблиц со статистикой.

Минусы использования

  • Использование элемента <figure> не всегда целесообразно. Например,
    графический баннер не стоит размечать в <figure>. Используйте для этого просто
    <img>.

  • Internet Explorer до версии 8.0 включительно игнорирует тег <figure>, но
    отображает его содержимое. Также в этом браузере любые стили не применяются
    к элементу, пока он не создан через скрипт, как показано на рисунке.

  • Firefox полностью поддерживает этот тег начиная с версии 4.0, но версии 3.0
    и старше также корректно отображают содержимое тега.
  • Браузеры Firefox и Chrome автоматически добавляют к <figure> свойство margin
    со значением 1em 40px.
